# Complete human influenza B virus genomes

### Source of data

The datasets used here was produced for [Dudas _et al._ (2014)](https://academic.oup.com/mbe/article/32/1/162/2925578) with original sequence data derived from GISAID.


### Visualisation(s)

The code here shows how to plot tangled chains (tanglegrams with more than two trees) and trees with discrete values associated with tips.
